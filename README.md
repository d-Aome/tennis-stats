# Description
A stat tracker for Tennis. Whose purpose it to hopefully track the following statistics: first serves in, first serves won, Second serve win %, and Break points converted. Besides including player statistic tracking tennis-stats also having the functionality for clubs and organizers to create tennis tournament brackets with potential seeding. Which will include tracking for players who won games, number of sets won by specific players.
