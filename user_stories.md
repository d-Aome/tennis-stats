# User Stories:
As a Tennis Club officer, I want to build a bracket for a tennis tournament that evenly pairs opponents with players of similar skill level.

As a recreational Tennis player, I want to practice my serves by tracking my first and second serve percentage so I can win more tennis matches.

As a recreational Tennis player, I want to improve my return of serve game by tracking how many break points I get so I can win my matches more consistently.

As a Tennis Club officer, I want to build a ranking system for individual players.

# Exceptions:
A player hasn't been added to the database. When the brackets are made, they aren't placed in the tournament.

A player doesn't have any data on a specific swing, so when calculated, it returns a divide by zero error.
